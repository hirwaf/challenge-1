## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript.

Here is [APIs Documentation](https://documenter.getpostman.com/view/1032842/TzzEouY2)

## Setting Up

```bash
$ yarn install
$ cp .env.example .env
$ yarn run seed:run
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
